Api (FastApi) + desktop aplikacja (Qt)
Aplikacja webowa została napisana w FastAPI i wykorzystuje system autoryzacji oparty o JWT. Umożliwia logowanie użytkowników oraz zarządzanie listą dostępnych towarów, w tym ich pobieranie i aktualizację. Całość działa w środowisku Docker, co zapewnia łatwe uruchamianie i izolację środowiska.

Aplikacja desktopowa powstała w Pythonie z użyciem biblioteki Qt i posiada panel logowania, który łączy się z API w celu pobrania tokena JWT. Po autoryzacji użytkownik może wyświetlić listę towarów pobranych z API, edytować je w interfejsie graficznym oraz przesyłać zmiany z powrotem do API.
